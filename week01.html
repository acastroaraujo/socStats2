The variable T depicts whether someone got the ‚Äútreatment‚Äù or not.

  • Create a new variable called Y that contains the observed outcomes.

  • What is the Average Treatment Effect (ATE) for this 8 person experiment?

Note. I‚Äôm only asking for a simple difference in means; this is also sometimes called the ‚Äúnaive estimator‚Äù (or ‚Äúnaive comparison‚Äù), which works just fine in simple experimental settings.

1.3 Exercise

  • Simulate a new completely randomized experiment on these 8 people; that is, re sample \(T\) at random so that equal numbers get the treatment and the control.
  • Create a new variable called Y that contains the observed outcomes.
  • What is the Average Treatment Effect (ATE) for this 8 person experiment?
  • Do this a couple of times (at least 3) and note the differences.

Hint: You can re-sample \(T\) very easily using the sample() function.

For example:

Code 
sample(d$T)
[1] 1 0 1 0 0 1 0 1

How do these estimates compare to the ‚Äúreal‚Äù ATE?

When I say ‚Äúthe real ATE‚Äù I basically mean this:

Code 
mean(d$Y1 - d$Y0)
[1] 0.125

1.4 Exercise

  • Assuming the ATE is \(0.125\), how many people would you need to enroll in this experiment to have enough statistical power?

The following data frame contains the potential outcomes for 8 individuals.

```{r}
d <- data.frame(
  T = c(0, 0, 1, 0, 0, 1, 1, 1),
  Y0 = c(5, 8, 5, 12, 4, 8, 4, 9),
```

The variable `T` depicts whether someone got the "treatment" or not.

::: callout-note

:::

*Note. I'm only asking for a simple difference in means; this is also sometimes called the "naive estimator" (or "naive comparison"), which works just fine in simple experimental settings.*
